# mattthorningphotography

A Sapper app with a Strapi API - hosted at [mattthorningphotography.com](https://matthorningphotography.com)

Requires .env file:

- PUBLIC_URL
- BE_CONTAINER_NAME
- FE_CONTAINER_NAME
- MONGO_CONTAINER_NAME
- API_URL
- APP_NAME
- ADMIN_JWT_SECRET
- EMAIL_ADDRESS
- EMAIL_USERNAME
- EMAIL_PASSWORD
- PAYPAL_ID
